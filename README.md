# 🕹️ Othello (Reversi) Web Application

A modern, responsive, and fully-featured Othello (Reversi) game built using **HTML, CSS, and JavaScript**.

This project supports both **Player vs Player** and **Player vs AI** modes with multiple difficulty levels.

## 🚀 Features

### 🎮 Game Modes
- Player vs Player (Local Multiplayer)
- Player vs Computer (AI)
  - Easy (Random moves)
  - Medium (Greedy strategy)
  - Hard (Minimax AI)

### 🧩 Core Gameplay
- Standard 8×8 board
- Valid move highlighting
- Automatic disc flipping
- Turn indicator
- Real-time score display
- Game over detection
- Restart without refresh

### 🎨 UI & Design
- Clean and modern interface
- Responsive for mobile & desktop
- Smooth animations
- Professional typography
- No blinking cursor / text selection

### ⚙️ Technical
- No external paid libraries
- Modular JavaScript logic
- High performance
- Cross-browser compatible
- Fully client-side
